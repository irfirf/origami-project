# origami-project

A program that takes your drawings and visualizes the folds that a sheet of paper needs to have for a single, straight cut to create the shape that was drawn. <br>
Inspired by [JOrigami](https://jorigami.sourceforge.net/). <br>
Resources: [Recent Results in Computational Origami (Demaine and Demaine)](https://erikdemaine.org/papers/OSME2001/paper.pdf), [A Computational Algorithm for Origami Design (Lang)](https://courses.csail.mit.edu/6.885/fall04/erik_notes/anydpi/L18_paper.pdf)
